### 정렬된 배열
* 삽입 전에 검색을 먼저 수행하고 올바른 위치를 찾은 다음 삽입한다.

```java
Integer sortedArrayLinearSearch(Integer[] array, int searchValue) {
    for (int result : array) {
        if (result == searchValue) {
            return result;
        } else if (result > searchValue) {
            break;
        }
    }
    return null;
}
```

### 삽입 정렬
1. 첫번째 패스스루에서 인덱스 1의 값을 삭제하고 임시변수에 저장한다 인덱스 1에는 공백이 생긴다.
2. 공백 왼쪽에 있는 각 값을 가져와 임시변수에 있는 값과 비교하는 시프트 단계를 시작한다.
3. 임시로 제거한 값을 현재 공백에 삽입한다.
4. 배열의 마지막 인덱스에서 패스스루를 시작할 때 까지 반복한다.

### 효율성
* N^2 + 2N - 2단계
* O(N^2 + N)
* 다양한 차수가 한데 섞여 있을 때, 빅오 표기법은 가장 높은 차수의 N만 고려한다.
* 즉, O(N^2)

### 평균적인 경우
* 최악의 시나리오
  * N^2
  * 선택 정렬 > 삽입 정렬
* 평균 시나리오
  * N^2 / 2 단계
* 최선의 시나리오
  * N
* 선택 정렬은 모든 시나리오에서 N^2 / 2
